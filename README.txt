项目名称：研料
描述：研料是一个考研二手资料交易平台，属于web应用。你可以在上面进行注册、登录，购买自己所需的考研资料，也可以在该平台进行售卖。

功能实现：
1.卖家：注册；登录登出；上架资料；更新资料
（卖家无法购买自己的资料，只能对资料进行信息更新，但同样可以作为买家可以购买其他人的资料）
2.买家：注册；登录登出；将资料加入购物车；将资料从购物车中删除；对卖家资料进行评论留言，留言板块显示留言者及时间信息可以删除；
  关键词搜索功能，将资料关键词输入即可搜索到相应的考研资料；
  
3.资料：首页显示所有资料基本信息，可以点击详情可以查询留言或添加留言，买家可以加入购物车
  当买家将资料加入购物车后，该资料将会显示已售出，无法继续加入购物车；
  当买家在购物车删除该商品后，卖家可以进行商品状态信息更新（从“已售出”变为“售卖中”），重新将商品上架进行售卖；
  同一件商品，对于卖家会显示“更新商品信息按钮”，对于买家会显示“加入购物车”按钮；
  购物车内能点进用户名字，看他上架的所有书目；同样可以浏览书目的所有信息，包括其他人对它的留言评价，符合用户浏览习惯

技术实现：
后台：php
前台：bootstrap,jQuery,css

开发环境：
操作系统：Windows10
编程语言：php
开发工具：sublime
数据库：MYSQL
代码托管平台：Github
